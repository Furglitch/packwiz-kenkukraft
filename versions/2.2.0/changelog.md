## The Overhaul Update!

Full Changelog: [*2.1.2* ➜ *2.2.0*](https://github.com/Furglitch/packwiz-kenkukraft/compare/2.1.2...2.2.0)

<ins>**Overhaul:**</ins>
- Changed Pack name to "KenkuKraft: Infected"
- Changed Pack assets to reflect new theme

<ins>**Configs Changed:**</ins>
- ***Bagus Lib***
    - Disabled 'Cooler End Poem' feature as it caused the new custom End Poem to not display
- ***Carry On***
    - Disabled carrying *Spore* mobs, Bosses, and *Lootr* containers
- ***Configured Defaults***
    - Enabled resource packs by default
    - Fixed keybinds
- ***FancyMenu***
    - Adjust main menu layout due to overlap from *Modpack Update Checker*
    - Adjust inventory button layout due to overlap from *Dragon Survival*
- ***Fancy Toasts***
    - Disabled the 'Regions Explored' advancement toast since it's likely triggered on spawn
    - Disable toasts for individual Quest tasks (not implemented, future update: Bivrik/FancyToasts#52)
    - Added custom toast texture, though it's disabled as it crashes the game on Linux (see Bivrik/FancyToasts#53), which is what my friend group primarily uses
- ***FTB Quests***
    - Added quests (Note: Incomplete, more to come)
- ***KubeJS***
    - Disabled /locate in favor of using *Explorer's Compass* and *Nature's Compass*

<ins>**Added:**</ins>
- *Create: Extended Wrenches*
- *Customizable Player Models*
- *FancyMenu*
- *Figura*
- *Modpack Update Checker*

<ins>**Removed:**</ins>
- *Better Fps - Render Distance[Forge]*: Redundant with other mods
- *Chunk Sending [Forge/Fabric]*: Redundant with other mods
- *Create: Cobblestone*: Redundant with *Create: Mechanical Extruder*
- *Fast Async World Save [Forge/Fabric]*: Redundant with other mods
- *fix GPU memory leak [Forge/Fabric]*: Redundant with other mods
- *Mob Control*: Integrated with KubeJS
- *Server Performance - Smooth Chunk Save [Forge/Fabric]*: Redundant with other mods
- *Terralith*: World Generation performance
- *Village Spawn Point*: Removal promotes exploration

<ins>**Swapped:**</ins>
- *Advancements Reloaded* ➜ *Better Advancements*: AR had some visual bugs. Likely a conflict with other UI mods.
- *Nemo's Inventory Sorting* ➜ *Sophisticated Sorter*: Personal preference for sorting style
