## Installation

1.  Download **[Prism Launcher](https://prismlauncher.org/)**
2.  Download the latest version of KenkuKraft from the **[Releases]** page.
3.  Click **"Add Instance"** in the top left.
4.  Click **"Import"** on the left sidebar.
5.  **Select the downloaded KenkuKraft zip file**
6.  Click **"Okay"** and wait for the instance to download.

### Updating

1. Right click the KenkuKraft instance in Prism Launcher and select "Edit"
2. Click **"CurseForge"** on the left sidebar.
3. Click the **"Update from file"** button.
4. Select the newly downloaded KenkuKraft zip file.
5. Wait for the update to finish.
